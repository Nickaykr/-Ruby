Команда "addmoney"

Добавляет пользователю указанное количество валюты

Использование:

/addmoney (user) (amount) (reason)

user - имя пользователя

amount - количество валюты

reason(необязателтный параметр) - причина поощрения

Примечание:

Этой командой может пользоваться только хозяин сервера

Команда "balance"

Показывает ваш баланс или баланс другого пользователя

Использование:

/balance (user)

user - имя пользователя чей баланс вы хотите увидеть (опустите этот параметр, если хотите увидеть свой баланс)

Команда "clear"

Удаляет указанное количество сообщений

Использование:

/clear (count)

count - количество сообщений, которые вы хотите удалить

Примечание:

Этой командой может пользоваться только админестраторы сервера

Команда "createrole"

Сщздаёт своб роль за валюту

Использование:

/createrole (name) (color) (roleprice)

name - название своей роли

color - цвет отображения роли

roleprice - цена, за которую можно купить роль

Команда "interaction"

Взаимодействует с пользователем через гифку

Использование:

/interaction (user) (reaction)

user - имя пользователя кому вы хотите отпрвить гифку

reaction - какого типа гифку вы хотите отпрвить (ударить, пожать руку, Обнять, приветсвовать)
